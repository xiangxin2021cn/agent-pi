# v1.0.0 - Agent Pi Goal Loop Workbench

## Release Theme

Agent Pi V1.0 is the first major workbench release after the Craft Agents OSS fork. The core change is the Goal Loop upgrade: long-running document tasks now get a conservative self-review layer that checks the user's stated goal, required output files, output formats, workspace output directories, attachment/source evidence, previewable artifact contents, and verification signals before treating a task as complete.

This is a major step toward reliable long-form work. Agent Pi is moving beyond a chat shell into a document production workbench where the agent executes, the app records evidence, and the user can inspect, edit, export, and verify deliverables.

## Major Goal Loop Upgrade

- Added persisted goal state for long tasks, including audit events, retry limits, stale recovery, manual controls, and UI status visibility.
- Added hidden goal continuation so unfinished tasks can continue without fake user messages polluting the conversation.
- Added deterministic checks for explicit output files, expected output folders, requested formats, attachments, source files, tool verification, and shallow completion claims.
- Added artifact preview evidence for readable text, Markdown, spreadsheets, Office conversions, and text-extractable PDFs so the reviewer can inspect actual outputs instead of relying only on the final assistant message.
- Added source/output distinction in audit evidence, reducing the risk that the model treats input materials as finished deliverables.
- Added citation and explicit-item checks for tasks that require source grounding or clearly listed deliverables.

## Document Workbench Upgrade

- Added in-app PDF preview support and safer PDF preview behavior in packaged Windows builds.
- Added Markdown preview export to PDF and DOCX using rendered Markdown output rather than raw Markdown source.
- Added native save dialogs for Markdown download and PDF/DOCX export.
- Added editable Markdown file preview for real `.md` and `.markdown` files.
- Fixed Markdown edit mode to preserve raw source content. Complex Markdown tables, Mermaid blocks, and long document bodies now stay intact when switching from preview to edit.
- Added bounded previews for Office and spreadsheet files so common project artifacts can be inspected without leaving the app.

## Core Runtime

- Includes Claude Agent SDK `0.3.195`.
- Includes Pi runtime packages `0.80.2`.
- Keeps the Windows package path focused on local productivity: Bun runtime, Claude native binary, ripgrep, helper servers, and Pi agent server resources are staged into the installer.

## Product Direction

- V1.0 positions Agent Pi as a super workbench for long-cycle work: tenders, contracts, construction plans, BOQ/Excel analysis, technical reports, research notes, and enterprise document production.
- The goal is not to make an uncontrollable autonomous system. The goal is observable, reviewable, and correctable long-task execution with stronger output quality guarantees.
