# v0.10.10 - Prompt and Document Preview Upgrade

## Features

- Added an Optimize Prompt button in the chat composer. It rewrites the current draft into clearer agent instructions before sending, using the active session model when available and a local fallback template when needed.
- Added editable Markdown file preview for real `.md` / `.markdown` files, including save-back with disk-change protection.
- Added Markdown export actions for PDF and DOCX from the Markdown preview overlay.
- Added read-only spreadsheet preview for `.xlsx`, `.xls`, and `.xlsm` files with bounded sheet, row, and column rendering.
- Added PDF preview zoom controls, zoom reset, and page count display.

## Improvements

- File preview RPC now returns source modification time for safer Markdown edits.
- Spreadsheet attachment previews continue to use bounded parsing safeguards for large or polluted workbook ranges.
- Registered the existing attachment dialog handler in handler coverage metadata.
- Improved Chinese and other IME input stability in the chat composer by keeping composition text visible, skipping draft synchronization while composition is active, and removing synchronous editor setup from the focus path.
- Strengthened Electron RPC handler registration tests so core/gui channel checks do not share captured state and have enough timeout margin under combined test runs.

## Validation

- Passed targeted prompt optimizer, spreadsheet preview/export, Office preview, file attachment dialog, file classification, routing, i18n parity, i18n sorting, handler registration, and related package typechecks.

## Breaking Changes

- None expected.
