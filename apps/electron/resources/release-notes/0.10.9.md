- Upgraded the Pi SDK packages from `@mariozechner/*` 0.73.1 to `@earendil-works/*` 0.79.9 while keeping the newer Claude Agent SDK 0.3.191 runtime from v0.10.8.
- Pulled in the upstream z.ai / GLM-5 catalog refresh through the newer Pi SDK, including GLM-5 family availability and updated reasoning metadata.
